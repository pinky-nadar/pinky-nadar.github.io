Behind the Design

I started my career from being a Front-end developer where I have learned solid technical skills. I transitioned to UX Design in 2019 where I got a chance to design the interface for an building inspection reporting application. It motivated me to create usable designs for the end-users. From then, I've been working on the human-centered design principles to create user interfaces with good user experience.
