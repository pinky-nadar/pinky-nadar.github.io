Behind the Design

  • ✨ Crafting usable interfaces and experience since 2019
  • ✨ Guiding young designers to create a user-friendly interface
  • ✨ Expert in Front-end Development
  • ✨ Engineer turned Designer

I started my career from being a Front-end developer where I have learned solid technical skills. I transitioned to UX Design in 2019 where I got a chance to design the interface for an web application. It motivated me to create usable designs for the end-users. From then, I've been working on the human-centered design principles to create user interfaces with good user experience.
